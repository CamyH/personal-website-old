This is a simple loan calculator that I made in Python using the tkinter library to create the GUI.
To use this calculator you simply input the interest rate, the number of payments you would like to make and the amount you are borrowing.
You can then calculate your monthly payment by clicking the corresponding button.
If you want you can then also display the remaining loan amount by clicking the button.
